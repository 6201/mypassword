# MyPassword Desktop 主要功能说明（用于 Stitch 生成 Mobile UI）

## 1. 产品目标
- 本地优先的密码管理器。
- 核心任务：查看、搜索、新增、编辑、删除密码条目；分类管理；安全锁定；备份与恢复。
- 安全原则：密码明文按需解密、可锁屏、支持自动锁定。

## 2. 核心数据模型
每条密码记录字段：
- `id: number`
- `title: string`（站点/条目标题）
- `username: string`
- `password: string`（详情页按需加载）
- `url?: string`
- `notes?: string`
- `category?: string`（默认 `Default`）
- `tags?: string`（逗号分隔）
- `favorite?: boolean`
- `updatedAt?: number`

## 3. 主要信息架构（Desktop 映射）
Desktop 是“三栏 + 顶栏动作”，Mobile 建议拆为多屏：

### 3.1 锁屏页（Lock Screen）
- 显示：应用已锁定提示 + 密码输入框 +「解锁」按钮。
- 失败反馈：
  - 密码错误
  - 设备密钥不可用
  - 请输入锁屏密码

### 3.2 主页面（Vault Home）
- 顶栏：
  - 搜索框
  - 安全设置按钮
  - 立即锁定按钮（仅已设置锁屏密码时显示）
  - 备份与恢复按钮
  - 添加密码按钮
  - 生成密码按钮
- 左栏（分类）：全部 / Default / 自定义分类
- 中栏（列表）：条目卡片（图标、标题、用户名、收藏星标、分类）
- 右栏（详情）：选中条目的完整信息 + 操作按钮（编辑、删除、复制、显示密码）

## 4. 关键功能

### 4.1 搜索与过滤
- 搜索字段：`title`、`username`。
- 分类过滤与搜索组合生效。
- 当当前选中条目不在过滤结果中时，自动切换到第一条结果。

### 4.2 分类管理
- 支持：新增、重命名、删除分类。
- `All` 不可编辑/删除。
- `Default` 不可删除（删除其他分类时条目迁移到 `Default`）。

### 4.3 条目列表与详情
- 列表项支持 favicon（失败回退为字符图标）。
- 详情支持：
  - 复制账号
  - 显示/隐藏密码
  - 复制密码
  - 打开网址
  - 显示分类、标签、备注、最后修改时间
  - 编辑、删除

### 4.4 新增/编辑密码（Password Form）
字段：
- 标题（必填）
- 用户名（必填）
- 密码（必填）
- 确认密码（必填）
- 网址
- 分类（下拉）
- 标签
- 备注
- 收藏开关

校验规则：
- 密码与确认密码必须一致。
- 提交按钮在密码不完整/不一致时禁用。

关闭规则（重点）：
- **点击遮罩层不能关闭新增/编辑弹框**。
- 只能通过：右上角关闭按钮 或 底部“取消”按钮。

### 4.5 密码生成器
配置项：
- 长度（4~64）
- 大写字母
- 小写字母
- 数字
- 特殊字符
- 排除模糊字符（0/O/o/l/1/I）
- 每类至少一个

功能：
- 生成密码
- 强度显示（弱/中/强）
- 复制密码
- 回填到表单（“使用此密码”）

### 4.6 备份与恢复（Export/Import）
三个 Tab：
1) 导出数据
- 输入加密密码 + 确认密码
- 密码长度至少 4 位
- 成功后展示导出条数与路径

2) 导入数据
- 输入备份密码
- 重复处理策略：`skip` / `overwrite` / `rename`
- 导入结果展示：导入/更新/跳过统计

3) 1Password 导入
- 支持：`1PUX`、`CSV`、`1PIF`
- 重复条目默认跳过
- 导入后展示统计

### 4.7 安全设置
- 设置/修改锁屏密码（修改时需输入当前密码）
- 启用/关闭自动锁定
- 空闲时长：1/5/10/15/30/60 分钟
- 支持“立即锁定”

自动锁定触发：
- 在开启自动锁定且当前未锁定时，监听用户活动；超时后自动锁定。

## 5. 状态与反馈规范
- 异步操作按钮显示 loading（如“导出中...”“导入中...”）。
- 成功/失败均需要醒目反馈（绿色/红色提示块）。
- 删除操作需要二次确认。
- 复制行为可显示“已复制”，并在一段时间后恢复。

## 6. Mobile UI 生成建议（给 Stitch）
建议映射为以下移动端页面：
1. `LockScreen`
2. `VaultListScreen`（列表 + 搜索 + 顶部动作入口）
3. `EntryDetailScreen`
4. `EntryFormScreen`（新增/编辑，整页而非弹窗）
5. `PasswordGeneratorSheet`（底部抽屉或全屏）
6. `BackupRestoreScreen`（Tab：导出/导入/1Password）
7. `SecuritySettingsScreen`
8. `CategoryManageScreen`（新增/重命名/删除）
