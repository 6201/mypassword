# 密码管理工具开发计划

## 项目概述

开发一个安全、易用的桌面密码管理工具

## 核心功能

1. **密码存储与管理** - 安全加密存储密码条目
2. **自动填充** - 快速填入账号密码
3. **1Password 导入** - 支持从 1Password 导出数据
4. **密码生成器** - 自动生成高强度随机密码

## 技术栈

- **框架**: Electron + React + TypeScript
- **数据库**: SQLite + SQLCipher (加密)
- **加密**: AES-256-GCM + PBKDF2/Argon2

## 密码生成器设计

### 可配置规则

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 长度 | 密码位数 | 16 |
| 大写字母 | A-Z | ✓ |
| 小写字母 | a-z | ✓ |
| 数字 | 0-9 | ✓ |
| 特殊字符 | !@#$%^&* 等 | ✓ |
| 排除模糊字符 | 排除 0/O/l/1 等 | 可选 |
| 必须包含每类 | 每类至少一个 | ✓ |

### 生成算法

```
1. 根据用户选择的字符集构建候选池
2. 如果"必须包含每类"启用：
   - 先从每类字符中各随机选一个
   - 剩余位数从合并池中随机选择
3. 否则直接从合并池中随机选择
4. Fisher-Yates 洗牌算法打乱顺序
5. 返回生成的密码
```
